O nome do projeto é WORKWELL. Ele Consiste em um sistema que usa um chatbot e também a gamificação para promover, e recompensar ativamente o autocuidado e a saúde mental no ambiente de trabalho. O principal sistema é o de pontuação, onde o funcionário pode registrar suas emoções e posteriormente usar seus pontos para ganhar certos beneficios na empresa (ex: acompanhamento pscicológico, vale-almoço, cursos grátis, etc.), incentivando um ciclo positivo de trabalho onde todos possam se sentir seguros. Nesse Website foi desenvolvido um protótipo de como funcionaria o sistema de recompensas com interatividade visual com o usuario, e principalmente apresentar o projeto no geral.

Dentre as tecnologias utilizadas:

* Estrutura do HTML5 com divs, classes, containers, footer, dentre outros. 
* Estilização dos elementos via CSS3, como flexbox, media querie para grande parte dos dispositivos, animações com hover, menu hamburguer, dentre outros.
* Funcionalidades backend via Java-script, tais como modal, alert, responsividade, validação de dados, manipulação do DOM, dentre outros.
